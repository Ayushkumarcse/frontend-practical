## ğŸ“š Tasks and Features

### Task 1: HTML Lists, Tables, and Navigation
- **Files**: `task1-lists-tables.html`, `task1-home.html`, `task1-about.html`, `task1-services.html`, `task1-contact.html`
- **Features**:
  - Unordered, ordered, and nested lists
  - HTML tables with colspan and rowspan
  - Education details in tabular format
  - Multi-page navigation with links

### Task 2: Login Form with Profile Navigation
- **Files**: `task2-login.html`, `task2-profile.html`
- **Features**:
  - Login form with username and password
  - Navigation to profile page on submission
  - Profile page displaying user information

### Task 3: Registration Form and Frameset
- **Files**: `task3-registration.html`, `task3-frameset.html`, `task3-*.html`
- **Features**:
  - Registration form with validation
  - Frameset with header, navigation, content, and footer sections
  - Multiple frame pages for navigation

### Task 4: Image Map with Hotspots
- **File**: `task4-imagemap.html`
- **Features**:
  - Image map with clickable hotspots
  - Interactive regions showing information
  - SVG overlay for better compatibility

### Task 5: Text Counter
- **File**: `task5-textcounter.html`
- **Features**:
  - Multi-line text input
  - Character, line, and word counting
  - Alert message with statistics

### Task 6: CSS Styling Examples
- **Files**: `task6-styling.html`, `task6-external.css`
- **Features**:
  - Inline, internal, and external CSS
  - Class, ID, and element selectors
  - Box model demonstration (margin, padding, border, content)
  - DHTML form validation

### Task 7: CSS Fonts and Background Images
- **File**: `task7-css-fonts-bg.html`
- **Features**:
  - Different font families (serif, sans-serif, monospace, cursive, fantasy)
  - Page and element background images
  - Font properties demonstration

### Task 8: Flexbox, Grid, and Responsive Design
- **File**: `task8-flexbox-grid.html`
- **Features**:
  - CSS Flexbox layouts
  - CSS Grid layouts
  - Responsive design with media queries
  - Hover effects, transitions, and animations

### Task 9: Number to Words Converter
- **File**: `task9-number-to-words.html`
- **Features**:
  - Converts numbers (0-999) to words
  - Input validation
  - Error handling for out-of-range and invalid inputs

### Task 10: MobileMukti.com Website
- **File**: `task10-mobilemukti.html`
- **Features**:
  - Complete e-commerce website design
  - Product listings
  - Responsive layout
  - Interactive elements

### Task 11: Registration Form Validation
- **File**: `task11-validation.html`
- **Features**:
  - Name validation (alphabets only, min 6 characters)
  - Password validation (min 6 characters)
  - Email validation (standard pattern)
  - Phone number validation (10 digits)

### Task 12: DOM Manipulation and Events
- **File**: `task12-dom-events.html`
- **Features**:
  - Add, remove, and modify HTML elements
  - Mouse events (click, hover, enter, leave)
  - Keyboard events (keypress, keydown)
  - Event logging system

### Task 13: JavaScript Arrays, Objects & CSS Positioning
- **File**: `task13-arrays-positioning.html`
- **Features**:
  - Array operations (add, delete, search, update)
  - Object manipulation
  - CSS positioning (static, relative, absolute, fixed, sticky)

### Task 14: Dynamic Web Page
- **File**: `task14-todo-calculator.html`
- **Features**:
  - Todo list application
  - Calculator with basic operations
  - Dynamic content manipulation

### Task 15-25: React Application
- **Location**: `react-app/`
- **Features**:
  - **Task 15**: Basic React application structure
  - **Task 16**: Functional components using JSX
  - **Task 17**: State management with useState and props
  - **Task 18**: Event handling (button clicks, form events)
  - **Task 19**: Conditional rendering
  - **Task 20**: Lists and Keys with .map()
  - **Task 21**: Forms and controlled components
  - **Task 22**: useEffect hook for side effects
  - **Task 23**: React Router for navigation
  - **Task 24**: Text input, display, and toggle component
  - **Task 25**: Reusable Button component

## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Node.js and npm (for React application)
- A code editor (VS Code recommended)

### HTML/CSS/JavaScript Files

1. Clone or download this repository
2. Open any HTML file directly in your web browser
3. No additional setup required

### React Application

1. Navigate to the React app directory:
   ```bash
   cd react-app
   ```

2. Install dependencies (if not already installed):
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
