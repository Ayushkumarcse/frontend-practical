CSS Positioning Examples

Different CSS positioning values and their effects:

Static (default) - Normal flow
Relative - Offset from normal position
Absolute - Relative to parent container
Sticky - Sticks when scrolling
Fixed - Relative to viewport (always visible)

Positioning Types:

  • Static: Default positioning, follows normal document flow
  • Relative: Positioned relative to its normal position
  • Absolute: Positioned relative to nearest positioned ancestor
  • Fixed: Positioned relative to viewport, stays in place when scrolling
  • Sticky: Acts like relative until scrolled, then becomes fixed
